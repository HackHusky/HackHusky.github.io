Project Description

The Quadruped code name Husky I is a four legged crab like robot that used the raspberry pi 2 as it brain and main control center. In addition, Husky I also included a servo controller and motor driver for servo control and motor driving. The driving mode is a 4 wheeled drive system with each wheel power by a single motor.Similar to the walking mode, the driving mode current ability is to go forward,backward,turn left, and turn right. The idea behind the dual walking and driving is that to allow the robot ability to travel through multi terrain and obstacle. The driving mode will mostly be used for complicated elevated terrain such as stair that will be difficult to traverse on wheel. In contrast , the driving mode is mostly used on floor or tar. In essence, the driving mode is for terrain that easy to traverse but it would take too long and too much effort to traverse in walking mode.

Project Status: Complete.

Project Description

This is a homemade 2 axis Gimbal, pitch and yaw, power by 2 servo. The pitch is control by a HS-645MG Ultra Torque servo that had the capabilities to rotate 180 degree. The yaw direction is power by HS-785hb Winch servo that will allow the Gimbal to rotate 360 degree. This Gimbal is a sub-project that I handle to be mounted on our rover for the University Rover Challenge 2016 that happen in June. The Gimbal system also had it own innovative UI that can be control using either mechanical mouse or touch screen.
